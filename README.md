## 🖥프로젝트 소개
의대, 보건 계열 대학생들을 위한 보건 의료 용어 사전.
기존 의료 용어 사전은 발음을 알 수 없거나, 읽어주는 기능이 없었기에 tts로 해당 단어의 발음을 읽어주기 위해 개발하였습니다.
