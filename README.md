## 🖥프로젝트 소개
의료 종사자 및 의대, 보건 계열 대학생들이 의학 용어를 쉽게 검색하고, 해당 의학 용어의 발음과 함께 공부할 수 있도록 지원 해 주는 것을 목표로 개발하였습니다.
